## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Add layers

```bash
# add controller
$ nest g controller controllers/<module-name>

# add service
$ nest g service services/<module-name>

# add repostiory
# manually add repositories/<module-name>.repository.ts and imports/exports <module-name>.repository im repositories.module.ts

# add gateway
$ nest g gateway gateways/<module-name>

# add schema
# manually add schemas/<module-name>.schemas.ts and imports <module-name>.schemas im services.module.ts or repositories.module.ts

```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app with docker-compose

```bash
# Running Nest.js/mongoDB background with docker-compose
$ docker-compose -p stockcraft up -d

# Remove all containers created with docker-compose
$ docker-compose -p stockcraft down
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
